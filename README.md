Project focused on transforming structured data into RDF and exploring it through an interactive web interface with basic querying capabilities.

The main goal is to create a prototype of a linked data source for graphics card information that is machine-readable.

* **THE APP:** (https://gpu-info.streamlit.app/)

### Data sources:

* **Dataset:** [GPUs Specs From 1986 to 2026](https://www.kaggle.com/datasets/ellimaaac/gpus-specs-from-1986-to-2026)
* **Factchecking:** [TechPowerUp GPU Database](https://www.techpowerup.com/gpu-specs/)
